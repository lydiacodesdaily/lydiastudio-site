# Lydia Studio

Gentle tools for focus, time awareness, and learning.

## About

This is the landing page for Lydia Studio, showcasing projects designed to make time, focus, and structure feel more supportive. The tools are built to be calm and low-pressure, offering structure without demanding it.

## Projects

- **[FlowMate](https://flowmate.club/)** - A gentle focus timer with soft audio cues for time awareness
- **[Flow Club Companion](https://chromewebstore.google.com/detail/flow-club-companion-focus/ncjcggjgiajhmeifohinbeeeegdiaehc)** - A lightweight companion tool for Flow Club sessions
- **[Resources Library](https://resources.lydiastud.io/)** - A curated library of tools, practices, and supports that help when focus is hard

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the site.
